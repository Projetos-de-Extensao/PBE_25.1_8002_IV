### Caso de Uso 2: Gerar Relatório Socioeconômico
* **Atores**: Administrador
* **Descrição**:Produz relatórios analíticos com dados coletados
* **Pré-condições**:
   *  Mínimo de 50 cadastros completos
   *  Usuário com perfil administrativo
        
* **Fluxo Principal**:
   *  Administrador acessa módulo de relatórios
   *  Seleciona parâmetros(Faixa etária, Área geográfica, Indicadores específicos)
   *  Sistema processa dados
   *  Exibe resultados em Gráficos, Tabelas comparativas, Mapas de calor
   *  Administrador exporta em PDF/Excel
* **Pós-condição**:
   *  Relatório armazenado no sistema
   *  Disponível para consulta futura
